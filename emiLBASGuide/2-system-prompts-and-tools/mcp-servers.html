MCP servers


MCP (Model Context Protocol) servers are programs that expose capabilities to an AI model through a standardized interface.
They act as the “tool layer” of an Agentic system, the part that lets the model do things instead of just generating text.

MCP servers are not part of the model, they are not part of the system prompt, and are not a part of the Harness.

Most Harnesses can use MCP servers

MCP capabilities:

They provide three types of capabilities:

  • Tools : active operations the model can call (write files, query APIs, search GitHub).
  • Resources : passive read-only data (file contents, database schemas, calendars).
  • Application Prompts : prebuilt instruction templates for workflows (summarize meetings, plan trips).

Think of them like "servers" that connect to your AI and give them tools and/or knowledge on how to run the tools.

This should make you understand what is an MCP server ... there is an unrealengine mcp , a sqlmcp , a lot of mcp servers are out there , and it's a great way to give your model tools.

Sometimes you might want the MCP server to be in the same container as your harness.

So If you want to be SAFE ( depending on the MCP server ) you might want both the MCP server and harness to be isolated

Locally ran

They can be Local , to give access to a filesystem , or a specific program, duckduckgo-search capabilities , etc .

There is a "blender-mcp server" , and connecting your AI to it , lets it "control blender" trough the mcp server.

Local filesystem MCPs should be restricted to specific directories to prevent accidental deletions or malicious misuse.

List of MCP servers ( Remote and Local )

Here's a list of websites that you can find MCP servers:

  • Official MCP servers repo : reference implementations + community list, maintained by Anthropic/community
  • mcp.so : searchable directory, big catalog
  • MCPservers.org : searchable directory, big catalog
  • Smithery : directory + one-click install/config generator, many are hosted (remote) not just local
  • Glama MCP directory : has quality/safety scoring per server
  • PulseMCP : directory + newsletter tracking new servers
  • Composio : hosted MCP integrations for real SaaS tools (Gmail, Slack, Notion, etc.), auth handled for you

Remember, be careful, since it would be easy to hide something malicious in an MCP server.

Online MCP ( Remote )

There are online MCP servers , here's some examples ( they need an API key, keep it safe ):

  • Zapier : server with several MCP connections , a LOT of apps , and selection of access level
  • OttoKit : hosted MCP server with 1,000+ app integrations
  • n8n: open-source workflow automation tool that exposes your custom-built automations as MCP tools for your AI via webhooks.
